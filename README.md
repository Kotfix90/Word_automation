# Программа для тиражирования технических паспортов
---
## 1. Назначение:
  - Скрип предназначен для работы с техническими документами для изменения серийных номеров в автоматическом и ручном режимах на основе готовых шаблонов docx-файлов(технических паспортов) и xlcs-таблицами(таблиц, содержаших заводские номера), что значительно упрощает работу с документами.

## 3. Использование:
  - Файл "как запустить.txt" содержит пошаговое объясненние запуска скрипта

## 4. Результаты работы:
  - В результате в директории, выборанной для сохранения, будет получено две папки с новыми файлами имеющими необходимые данные (в виде порядковых серийных номеров и пр.) файлами: 
    "Для заказчика" и "Для заказчика заверенные"

  - В первой папке будут содержаться документы с удаленными изображениями подписей печатй штампов и т.д.
  > Примечание:
    Удаление необходимых изображений зависит от выбранных рядом с изображением строк, поэтому данная функция может сработать не корректно

  - Во второой папке будут содержаться документы, содержащие всё необходимое для элеткронной версии документа
